# phishingEmail 邮件发送工具

### 简介
phishingEmail 是一款命令行工具，用于测试SMTP服务器发送邮件，它的设计理念是小巧灵活易于使用，是实际运营中一款常用的全能邮件发送工具之一。
phishingEmail的优点：

    可以指定多个收件人，支持邮件批量发送；
    支持html邮件内容；
    支持附件发送；
    支持发件人伪造；
    支持无账户密码发送邮件；
    兼容性比较好，有windows exe版本；
### 参数说明
    -t 指定收件人
    -a 收件人地址文件
    -f 发送人邮箱
    -n 伪造发件人
    -j 主题
    -e 发送内容，内容为html文件
    --attach 指定附件
